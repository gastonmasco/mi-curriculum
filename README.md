# Mi Currículum Profesional - Gaston Masco

Este repositorio contiene mi currículum profesional interactivo y portafolio técnico, diseñado con una estética de **IDE Dashboard** premium.

## 🚀 Ver en vivo
Puedes acceder a la versión web aquí:
👉 **[https://gastonmasco.github.io/mi-curriculum/](https://gastonmasco.github.io/mi-curriculum/)**

## 🛠️ Tecnologías utilizadas
- **HTML5** (Semántico)
- **CSS3** (Variables, Flexbox, Grid, Animaciones)
- **JavaScript** (Efectos de escritura, detección de región, filtrado de portafolio)
- **GitHub Pages** (Despliegue)

## 📁 Estructura del Proyecto
- `index.html`: Currículum principal con experiencia, educación y habilidades.
- `portfolio.html`: Portafolio interactivo filtrable por categorías técnicas.
- `Perfil.png`: Foto de perfil profesional.
